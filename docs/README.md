# Документація платформи Pulito

Повна технічна та функціональна документація **Pulito** — гуртово-роздрібної e-commerce платформи (інтернет-магазин побутової хімії) на Next.js 16 з multi-tenancy, SaaS-білінгом, B2B-ціноутворенням та 20+ зовнішніми інтеграціями.

> Ця документація описує систему станом на актуальну гілку коду. Усі факти звірені безпосередньо з вихідним кодом у `src/`, `prisma/` та конфігурацією проєкту.

---

## Стек у двох словах

Next.js 16 (App Router, RSC, Server Actions) · React 19 · TypeScript 5.9 (strict) · Tailwind CSS 4 · PostgreSQL 16 + Prisma 7.4 + PgBouncer · Redis 7 (кеш / сесії / rate-limit / BullMQ) · Typesense (пошук) · JWT + Google OAuth + 2FA TOTP · LiqPay / Monobank / WayForPay · Нова Пошта / Укрпошта · Telegram · GA4 + Facebook CAPI · Sentry / Axiom · Cloudflare / R2 · AI (Claude + Gemini) · Vitest / Playwright / k6 · Docker / PM2 / Nginx.
