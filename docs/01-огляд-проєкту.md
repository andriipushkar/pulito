# 01. Огляд проєкту Pulito

Pulito — це enterprise-рівнева гуртово-роздрібна e-commerce платформа для продажу побутової хімії, побудована на Next.js 16. Це не просто інтернет-магазин, а повноцінна SaaS-система з multi-tenancy, B2B-ціноутворенням, понад 20 зовнішніми інтеграціями та майже повним автоматичним тестовим покриттям.

> Цей документ дає високорівневе розуміння «що вміє ця система». Технічні деталі стека — у [02-технології.md](02-технології.md), внутрішня будова — у [03-архітектура.md](03-архітектура.md).

---

## Що це за платформа

Pulito обслуговує одночасно **три різні бізнес-моделі** в одному коді:

1. **Роздріб (B2C)** — класичний інтернет-магазин: каталог, кошик, checkout, особистий кабінет, лояльність, реферали.
2. **Гурт / B2B (wholesale)** — три рівні гуртових цін, персональне ціноутворення per-client, volume-знижки, комерційні пропозиції (КП), кредитний ліміт, палетна доставка, персональний менеджер.
3. **SaaS / Multi-tenancy** — платформа здається в оренду іншим продавцям: ізоляція тенантів, subdomain/custom-domain routing, тарифні плани, біллінг, usage metering, white-label, feature flags.

Платформа орієнтована на український ринок: інтеграції з Новою Поштою та Укрпоштою, платіжними системами LiqPay / Monobank / WayForPay, маркетплейсами Prom.ua / Rozetka / OLX / Epicentr, обліковими системами 1С / BAS, а також підтримка ФОП-податкової звітності (касовий метод).

## Для кого ця система

| Роль                                 | Що отримує                                                                                                                                                          |
| ------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Роздрібний покупець**              | Швидкий каталог із fulltext-пошуком, зручний checkout (у т.ч. гостьовий), бонусні бали, купони, підписки Subscribe & Save, особистий кабінет із timeline замовлень. |
| **Гуртовий клієнт (B2B)**            | Спеціальні гуртові ціни (3 рівні), персональні ціни, bulk-замовлення через Excel/CSV, КП у PDF, кредитний ліміт, палетна доставка.                                  |
| **Власник магазину / адміністратор** | Повна адмін-панель: товари, замовлення, склади, аналітика (18 вкладок), звіти, маркетинг, інтеграції, постачальники.                                                |
| **Менеджер**                         | Обробка замовлень, ТТН, чат із клієнтами, модерація відгуків.                                                                                                       |
| **SaaS-орендар (tenant)**            | Власний субдомен/домен, тема оформлення, тариф, ізольовані дані.                                                                                                    |

## Ключові можливості (одним списком)

- **Каталог і пошук:** fulltext-пошук (Typesense, typo-tolerance, фасети), фільтри, сортування, бренди, категорійне дерево.
- **Картка товару:** zoom/lightbox-галерея, Quick View, floating buy bar, рекомендації «З цим купують», порівняння до 4 товарів, нещодавно переглянуті, бандли (набори зі знижкою).
- **Кошик і checkout:** real-time перерахунок, mini-cart, swipe-to-delete (mobile), 4-кроковий checkout + гостьовий, купони, бонусні бали, volume pricing.
- **Особистий кабінет:** замовлення з timeline статусів, PDF-рахунок, повторне замовлення, 2FA (TOTP + backup codes), історія входів, лояльність (4 рівні Bronze→Platinum, стріки, челенджі), реферальна програма, множинні вішлісти, bulk-order, підписки, прогнози покупок, прайс-листи PDF, GDPR self-delete + data export.
- **B2B / гурт:** 3 рівні гуртових цін, персональні ціни, volume pricing engine, КП у PDF, кредитний ліміт, палетна доставка, wholesale-правила.
- **Адмін-панель:** замовлення (фільтри, статуси, bulk, ТТН, PDF), товари (CRUD, watermark/WebP/AVIF, drag-and-drop, Excel import), аналітика (KPI, ABC, RFM, когорти, LTV, churn, forecast), 13+ типів звітів, маркетинг (банери, CMS, FAQ, блог, email-кампанії, публікації), multi-warehouse, audit log, live chat (WebSocket), теми, SEO-шаблони/аудит.
- **Постачальники та імпорт:** auto-sync цін/залишків із фідів (YML/CSV/XLSX/1С-ZIP/Google Sheets), консигнація + дропшип, націнка, резерв, звіт розрахунків.
- **Маркетплейси:** OLX/Rozetka через API, Prom/Epicentr через YML-фід.
- **Інтеграції:** платежі, доставка, боти, соцмережі, ERP, аналітика, моніторинг, CDN/storage, AI.
- **SaaS / multi-tenancy:** ізоляція тенантів, subdomain/custom-domain routing, тарифи, інвойси, usage metering, feature flags, white-label.
- **Безпека:** JWT RS256/HS256 з ротацією + reuse detection, rate limiting (Redis sliding window), CSRF, XSS (DOMPurify), 12 security headers (CSP+nonce, HSTS, COEP, COOP), idempotency, cookie consent, GDPR.
- **PWA:** Service Worker із 3-рівневим кешуванням, offline-режим, install prompt, push notifications.
- **SEO:** Schema.org (Product, FAQ, Organization, Breadcrumb), chunked sitemap, robots, canonical, hreflang (uk/en), Google Shopping XML feed, RSS, OG/Twitter Cards, ISR.
- **AI:** Anthropic Claude + Google Gemini (AI-описи, summary, класифікація відгуків, daily briefing).

## Технологічне резюме (коротко)

- **Frontend / Framework:** Next.js 16 (App Router, RSC, Server Actions, ISR), React 19, TypeScript 5.9 (strict), Tailwind CSS 4.
- **Дані:** PostgreSQL 16 + Prisma 7.4, Redis 7 (кеш / сесії / rate-limit / BullMQ-черги), Typesense (пошук).
- **Аутентифікація:** JWT RS256/HS256 (httpOnly cookies) + Google OAuth + 2FA (TOTP).
- **Тестування:** Vitest (unit/integration), Playwright (E2E + a11y + visual), k6 (load).

Повний стек із поясненнями «чому» і «де» — у [02-технології.md](02-технології.md).
