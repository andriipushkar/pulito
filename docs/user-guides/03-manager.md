# Інструкція для менеджера

Менеджер відповідає за щоденну операційну роботу магазину: обробку замовлень, управління товарами, роботу з клієнтами та модерацію контенту.

---

## 1. Вхід та дашборд

### 1.1. Вхід у панель управління

1. Перейдіть за адресою адмін-панелі (наприклад, `https://ваш-сайт/admin`).
2. Введіть **Email** та **Пароль** вашого менеджерського акаунту.
3. Якщо увімкнено **2FA**, введіть 6-значний код з додатку-автентифікатора.
4. Натисніть **Увійти**.

### 1.2. Дашборд

Після входу ви потрапляєте на головний дашборд. На ньому відображаються:

#### KPI картки

- **Нові замовлення** — кількість замовлень зі статусом "Нове" за сьогодні.
- **Замовлення в обробці** — кількість замовлень, що потребують дій.
- **Відправлені сьогодні** — кількість замовлень зі статусом "Відправлено" за сьогодні.
- **Виручка за сьогодні** — загальна сума оплачених замовлень.

#### Сповіщення про нові замовлення

- У верхньому правому куті відображається іконка дзвіночка з лічильником нових подій.
- Натисніть на дзвіночок, щоб побачити список нових сповіщень:
  - Нове замовлення (номер, сума, дата).
  - Новий оптовий запит.
  - Новий відгук на модерацію.
  - Нове звернення.
- Натисніть на сповіщення, щоб перейти до відповідного розділу.

---

## 2. Управління замовленнями

### 2.1. Список замовлень

1. У бічному меню оберіть **Замовлення**.
2. Відкриється таблиця з усіма замовленнями:

| Колонка | Опис |
|---------|------|
| **#** | Номер замовлення |
| **Дата** | Дата та час створення |
| **Клієнт** | Ім'я та прізвище покупця |
| **Телефон** | Контактний телефон |
| **Сума** | Загальна сума замовлення |
| **Статус** | Поточний статус (кольоровий бейдж) |
| **Оплата** | Статус оплати |
| **ТТН** | Номер товарно-транспортної накладної |
| **Дії** | Кнопки дій |

#### Пошук замовлень

У верхній частині списку знаходиться поле пошуку. Ви можете шукати за:
- **Номером замовлення** — введіть повний або частковий номер.
- **Ім'ям клієнта** — введіть ім'я або прізвище.
- **Телефоном** — введіть номер телефону (повністю або останні цифри).
- **ТТН** — введіть номер накладної.

#### Фільтри

Натисніть **Фільтри** для відкриття панелі фільтрації:
- **Статус** — оберіть один або кілька статусів (Нове, Підтверджено, Відправлено, Доставлено, Скасовано, Повернуто).
- **Дата** — вкажіть діапазон дат (від — до).
- **Спосіб оплати** — накладений платіж, онлайн, за реквізитами.
- **Спосіб доставки** — Нова Пошта, Укрпошта, Самовивіз.
- **Сума** — від і до.

Натисніть **Застосувати** для оновлення списку.

#### Сортування

Натисніть на заголовок будь-якої колонки для сортування:
- За датою (від нових / від старих).
- За сумою (від більшої / від меншої).
- За статусом.

### 2.2. Обробка замовлення

#### Перехід до деталей

1. Натисніть на номер замовлення або кнопку **Переглянути** (іконка ока).
2. Відкриється сторінка деталей замовлення.

#### Інформація на сторінці замовлення

- **Дані клієнта** — ім'я, телефон, email.
- **Адреса доставки** — тип, місто, відділення або адреса.
- **Товари** — таблиця з усіма позиціями (назва, кількість, ціна, сума).
- **Підсумок** — загальна сума, знижка, вартість доставки, до оплати.
- **Статус замовлення** — поточний статус та кнопка зміни.
- **Статус оплати** — оплачено / не оплачено / часткова оплата.
- **ТТН** — поле для номера накладної.
- **Коментарі** — внутрішні нотатки та коментарі клієнта.

#### Зміна статусу замовлення

Статуси та допустимі переходи:

```
Нове → Підтверджено → Відправлено → Доставлено
  ↓         ↓
Скасовано  Скасовано
```

1. На сторінці замовлення знайдіть блок **Статус**.
2. Натисніть на випадаючий список поточного статусу.
3. Оберіть новий статус (відображаються лише допустимі переходи):
   - **Нове → Підтверджено** — ви підтвердили замовлення, зв'язалися з клієнтом.
   - **Підтверджено → Відправлено** — товар відправлено (обов'язково вкажіть ТТН).
   - **Відправлено → Доставлено** — клієнт отримав товар.
   - **Нове → Скасовано** або **Підтверджено → Скасовано** — замовлення скасовано.
4. Натисніть **Зберегти**.
5. Клієнт отримає email-сповіщення про зміну статусу.

> **Увага:** Перехід зі статусу "Відправлено" назад у "Підтверджено" неможливий. Будьте уважні при зміні статусу.

### 2.3. ТТН (товарно-транспортна накладна)

#### Введення ТТН вручну

1. На сторінці замовлення знайдіть поле **ТТН**.
2. Введіть 14-значний номер накладної.
3. Система автоматично перевірить формат (14 цифр).
4. Якщо формат некоректний, ви побачите повідомлення **"ТТН повинна містити 14 цифр"**.
5. Натисніть **Зберегти**.

#### Створення ТТН через Нову Пошту

1. На сторінці замовлення натисніть кнопку **Створити ТТН** (доступна, якщо інтеграція з Новою Поштою налаштована).
2. Відкриється форма з автоматично заповненими даними:
   - Відправник — дані магазину.
   - Отримувач — дані клієнта з замовлення.
   - Вага та габарити — заповніть або скоригуйте.
3. Перевірте дані та натисніть **Створити**.
4. Номер ТТН автоматично заповниться у полі.
5. Клієнт отримає сповіщення з номером ТТН.

#### Валідація ТТН

- ТТН повинна складатися рівно з **14 цифр**.
- Система не дозволить зберегти некоректний номер.
- При збереженні ТТН система перевіряє формат та виводить повідомлення про помилку, якщо формат не відповідає.

### 2.4. Документи

На сторінці замовлення у блоці **Документи** доступні:

#### Рахунок PDF

1. Натисніть кнопку **Рахунок PDF**.
2. Система згенерує PDF-документ з:
   - Реквізитами магазину.
   - Даними замовлення (номер, дата).
   - Таблицею товарів (назва, кількість, ціна, сума).
   - Загальною сумою.
3. Файл автоматично завантажиться або відкриється в новій вкладці.

#### Накладна PDF

1. Натисніть кнопку **Накладна PDF**.
2. Система згенерує товарну накладну у форматі PDF.
3. Документ містить дані відправника, отримувача та перелік товарів.

### 2.5. Коментарі до замовлення

1. На сторінці замовлення знайдіть блок **Коментарі**.
2. Ви побачите:
   - **Коментар клієнта** — якщо покупець залишив примітку при оформленні.
   - **Внутрішні коментарі** — нотатки менеджерів (не видимі клієнту).
3. Щоб додати внутрішній коментар:
   - Введіть текст у поле **Новий коментар**.
   - Натисніть **Додати**.
4. Коментар з'явиться у хронологічному списку із зазначенням автора, дати та часу.

### 2.6. Відправка фото товару клієнту

1. На сторінці замовлення знайдіть кнопку **Надіслати фото**.
2. Натисніть на неї — відкриється діалогове вікно.
3. Натисніть **Обрати файл** та оберіть фото з вашого комп'ютера (JPG, PNG; максимум 5 МБ).
4. За бажанням додайте **текстовий коментар** до фото.
5. Натисніть **Надіслати**.
6. Клієнт отримає email з фотографією товару та вашим коментарем.
7. Фото також збережеться у коментарях до замовлення.

### 2.7. Масові операції

Масові операції дозволяють обробити кілька замовлень одночасно.

1. У списку замовлень поставте **галочки** ліворуч від потрібних замовлень.
2. Або поставте галочку у заголовку таблиці, щоб обрати всі замовлення на поточній сторінці.
3. Зверху або знизу таблиці з'явиться панель масових дій:

#### Масова зміна статусу

1. Оберіть замовлення.
2. У випадаючому списку **Дія** оберіть **Змінити статус**.
3. Оберіть новий статус.
4. Натисніть **Застосувати**.
5. Статус зміниться для всіх обраних замовлень (з урахуванням допустимих переходів).

#### Масовий друк

1. Оберіть замовлення.
2. Натисніть **Друк** — відкриється сторінка друку з рахунками/накладними для всіх обраних замовлень.

#### Масове введення ТТН

1. Оберіть замовлення.
2. Натисніть **ТТН** — відкриється форма для введення ТТН для кожного замовлення.
3. Введіть номери та натисніть **Зберегти все**.

#### Масовий експорт

1. Оберіть замовлення (або усі з фільтру).
2. Натисніть **Експорт**.
3. Оберіть формат: **Excel (.xlsx)** або **CSV (.csv)**.
4. Файл з даними замовлень завантажиться на ваш комп'ютер.

## 3. Товари

### 3.1. Перегляд та пошук

1. У бічному меню оберіть **Товари**.
2. Відкриється каталог товарів у вигляді таблиці:
   - **Зображення** (мініатюра).
   - **Назва**.
   - **SKU** (код товару).
   - **Ціна** (роздрібна).
   - **Залишок** на складі.
   - **Статус** (активний / неактивний).
3. Використовуйте поле **Пошук** для пошуку за назвою, SKU або описом.
4. Використовуйте **Фільтри** для звуження результатів за категорією, наявністю, статусом.

### 3.2. Редагування товару

1. Натисніть на назву товару або кнопку **Редагувати** (олівець).
2. Відкриється форма редагування з такими полями:

#### Основна інформація

- **Назва** — назва товару (обов'язково).
- **Опис** — повний опис (HTML-редактор з форматуванням).
- **Короткий опис** — відображається в каталозі.
- **SKU** — код товару.
- **Категорія** — оберіть із дерева категорій.
- **Бренд** — оберіть зі списку або додайте новий.

#### Ціна та залишок

- **Роздрібна ціна** — основна ціна товару.
- **Стара ціна** — ціна до знижки (перекреслена на картці).
- **Залишок** — кількість на складі. Введіть нове значення для оновлення.

#### SEO

- **Meta Title** — заголовок для пошукових систем.
- **Meta Description** — опис для пошукових систем.
- **URL slug** — частина URL-адреси товару.

3. Внесіть зміни.
4. Натисніть **Зберегти**.

### 3.3. Завантаження зображень

1. На сторінці редагування товару знайдіть блок **Зображення**.
2. Натисніть **Додати зображення** або перетягніть файли у зону завантаження.
3. Підтримувані формати: JPG, PNG, WebP.
4. Максимальний розмір файлу: 5 МБ.
5. Перше зображення автоматично стає головним.
6. Щоб змінити порядок — перетягніть зображення drag-and-drop.
7. Щоб видалити — натисніть **✕** на мініатюрі.
8. Натисніть **Зберегти**.

### 3.4. Імпорт з Excel/CSV

1. У розділі **Товари** натисніть кнопку **Імпорт**.
2. Завантажте файл у форматі **.xlsx** або **.csv**.
3. Формат файлу:
   - Перший рядок — заголовки колонок.
   - Обов'язкові колонки: **SKU**, **Назва**, **Ціна**.
   - Додаткові колонки: **Опис**, **Категорія**, **Залишок**, **Стара ціна**, **Meta Title**, **Meta Description**.
4. Натисніть **Завантажити**.
5. Система покаже попередній перегляд даних та кількість товарів для імпорту.
6. Перевірте дані та натисніть **Імпортувати**.
7. Після завершення ви побачите звіт:
   - Кількість створених нових товарів.
   - Кількість оновлених товарів (за SKU).
   - Кількість помилок (якщо є) з деталями.

## 4. Клієнти

### 4.1. Перегляд списку клієнтів

1. У бічному меню оберіть **Клієнти**.
2. Відкриється таблиця з усіма зареєстрованими користувачами:
   - **Ім'я**.
   - **Email**.
   - **Телефон**.
   - **Роль** (Клієнт / Оптовий покупець).
   - **Дата реєстрації**.
   - **Кількість замовлень**.
   - **Загальна сума покупок**.

#### Пошук клієнтів

Використовуйте поле пошуку для знаходження клієнта за:
- Ім'ям або прізвищем.
- Email-адресою.
- Номером телефону.

### 4.2. Підтвердження оптових запитів

1. У розділі **Клієнти** перейдіть на вкладку **Оптові запити** або зверніть увагу на сповіщення на дашборді.
2. Ви побачите список запитів зі статусом **Очікує розгляду**.
3. Натисніть на запит для перегляду деталей:
   - Дані клієнта (ім'я, email, телефон).
   - Назва компанії, ЄДРПОУ.
   - Сфера діяльності.
   - Очікуваний обсяг закупівель.
   - Коментар клієнта.
4. Прийміть рішення:
   - **Підтвердити** — натисніть кнопку **Підтвердити** та оберіть оптову групу (1, 2 або 3). Клієнт отримає email-сповіщення.
   - **Відхилити** — натисніть **Відхилити** та вкажіть причину. Клієнт отримає email з поясненням.

## 5. Відгуки

### 5.1. Модерація відгуків

1. У бічному меню оберіть **Відгуки**.
2. За замовчуванням відкривається вкладка **На модерації** — відгуки, що очікують перевірки.
3. Для кожного відгуку відображається:
   - **Автор** — ім'я та email клієнта.
   - **Товар** — назва товару (клікабельна).
   - **Рейтинг** — зірочки від 1 до 5.
   - **Текст** відгуку.
   - **Дата** створення.

### 5.2. Публікація відгуку

1. Прочитайте текст відгуку.
2. Якщо відгук відповідає правилам (не містить спам, ненормативну лексику, не є фейковим):
3. Натисніть кнопку **Опублікувати** (зелена галочка).
4. Відгук стане видимим на картці товару для всіх відвідувачів.

### 5.3. Відхилення відгуку

1. Якщо відгук порушує правила:
2. Натисніть кнопку **Відхилити** (червоний хрестик).
3. У діалоговому вікні оберіть або введіть **причину відхилення**:
   - Спам або реклама.
   - Ненормативна лексика.
   - Не стосується товару.
   - Інша причина (введіть вручну).
4. Натисніть **Підтвердити відхилення**.
5. Автор отримає email з причиною відхилення.

### 5.4. Відповідь на відгук

1. Натисніть кнопку **Відповісти** під відгуком.
2. Введіть текст відповіді від імені магазину.
3. Натисніть **Надіслати**.
4. Відповідь з'явиться під відгуком на картці товару з позначкою **"Відповідь магазину"**.

## 6. Зворотній зв'язок

### 6.1. Перегляд звернень

1. У бічному меню оберіть **Зворотній зв'язок** або **Звернення**.
2. Відкриється список звернень від клієнтів:
   - **Ім'я** відправника.
   - **Email**.
   - **Тема** звернення.
   - **Дата**.
   - **Статус** (Нове / В роботі / Закрите).

### 6.2. Обробка звернення

1. Натисніть на звернення для перегляду деталей.
2. Прочитайте текст повідомлення клієнта.
3. Підготуйте відповідь:
   - Введіть текст у поле **Відповідь**.
   - Натисніть **Надіслати відповідь**.
4. Клієнт отримає вашу відповідь на email.
5. Змініть статус звернення:
   - **В роботі** — якщо питання потребує додаткового часу.
   - **Закрите** — якщо питання вирішено.
6. Натисніть **Зберегти**.

## 7. Telegram/Viber сповіщення

### 7.1. Які сповіщення отримує менеджер

Менеджер автоматично отримує сповіщення у підключений месенджер (Telegram або Viber) про такі події:

| Подія | Приклад повідомлення |
|-------|---------------------|

### 7.2. Налаштування сповіщень

Підключення месенджерів здійснюється адміністратором у налаштуваннях системи (див. [04-admin.md](./04-admin.md), розділ 3.5). Менеджер може попросити адміністратора:
- Підключити свій Telegram-акаунт для отримання сповіщень.
- Налаштувати типи сповіщень (які події надсилати).

> **Примітка:** Для роботи сповіщень потрібно, щоб ви написали боту магазину в Telegram та натиснули **Start**.
