# 08 — Адмін: аналітика, звіти, PDF

---

## 8.1 Дашборд аналітики

### TC-AA-1.1 Головний дашборд
**Передумова:** `/admin/analytics`
**Кроки:**
1. Перевірити KPI картки: виручка, замовлення, середній чек, клієнти
2. Перевірити графіки: динаміка продажів
3. Змінити період: 7 днів → 30 днів → 90 днів
**Очікуваний результат:** Дані оновлюються при зміні періоду
**Статус:** ⬜

---

## 8.2 Модулі аналітики

### TC-AA-2.1 ABC-аналіз
**Передумова:** `/admin/analytics` → ABC
**Кроки:**
1. Перевірити розподіл товарів по групах A, B, C
2. Перевірити таблицю товарів
**Очікуваний результат:** Класифікація товарів за важливістю
**Статус:** ⬜

### TC-AA-2.2 Географія
**Кроки:**
1. Перейти на модуль "Географія"
2. Перевірити: міста, кількість замовлень, виручка
**Очікуваний результат:** Таблиця міст з метриками
**Статус:** ⬜

### TC-AA-2.3 RFM сегментація
**Кроки:**
1. Перейти на "RFM"
2. Перевірити сегменти: Champions, Loyal, At Risk, Lost
**Очікуваний результат:** Клієнти розподілені по сегментах
**Статус:** ⬜

### TC-AA-2.4 Воронка продажів
**Кроки:**
1. Перейти на "Воронка"
2. Перевірити етапи: перегляди → кошик → замовлення → оплата
**Очікуваний результат:** Візуалізація конверсії між етапами
**Статус:** ⬜

### TC-AA-2.5 LTV клієнтів
**Кроки:**
1. Перейти на "LTV"
2. Перевірити: середній LTV, топ клієнтів, розподіл
**Очікуваний результат:** Дані про lifetime value клієнтів
**Статус:** ⬜

## 8.3 Звіти

### TC-AA-3.1 Генерація PDF звіту
**Передумова:** `/admin/reports`
**Кроки:**
1. Обрати шаблон: "Звіт про продажі"
2. Вказати період: останній місяць
3. Обрати формат: PDF
4. Натиснути "Генерувати"
5. Завантажити файл
6. Відкрити PDF і перевірити:
   - Синій хедер з назвою компанії
   - Заголовок звіту з датою
   - Таблиця з даними (колонки рівномірно)
   - Футер з контактами
**Очікуваний результат:** Гарний PDF з фірмовим стилем
**Статус:** ⬜

### TC-AA-3.2 Генерація Excel звіту
**Кроки:**
1. Обрати шаблон: "Залишки товарів"
2. Формат: XLSX
3. Генерувати → завантажити → відкрити
**Очікуваний результат:** Excel файл з даними
**Статус:** ⬜

### TC-AA-3.3 Всі типи звітів (PDF)
Перевірити кожен тип — PDF завантажується і читабельний:
- ⬜ Звіт про продажі
- ⬜ Залишки товарів
- ⬜ Замовлення за статусом
- ⬜ Активність клієнтів
- ⬜ Оптові продажі
- ⬜ Звіт по доставках
- ⬜ Фінансовий звіт
- ⬜ Повернення та скасування
- ⬜ Оптові групи
- ⬜ Товари-лідери
- ⬜ Активність менеджерів
- ⬜ Канали залучення
- ⬜ Зведений звіт

### TC-AA-3.4 Аналітичний PDF
**Передумова:** `/admin/analytics` → "Експорт PDF"
**Кроки:**
1. Натиснути "Експорт PDF" на будь-якому модулі
2. Завантажити і відкрити
**Очікуваний результат:** PDF з KPI, таблицями, branded header/footer
**Статус:** ⬜

## 8.4 Прайс-лист

### TC-AA-4.1 Генерація роздрібного прайсу
**Кроки:**
1. `/admin` → "Прайс-лист" → Роздрібний
2. Завантажити PDF
3. Перевірити: хедер, таблиця (код, назва, категорія, ціна, наявність)
**Очікуваний результат:** Гарний PDF прайс з повною шириною таблиці
**Статус:** ⬜

### TC-AA-4.2 Генерація оптового прайсу
**Кроки:**
1. Прайс-лист → Оптовий
2. Перевірити що ціни оптові
**Очікуваний результат:** PDF з оптовими цінами
**Статус:** ⬜

### TC-AA-4.3 Ілюстрований каталог
**Кроки:**
1. "Каталог товарів" (ілюстрований)
2. Перевірити: титульна сторінка, зміст, товари по категоріях
**Очікуваний результат:** PDF каталог з цінами та описами
**Статус:** ⬜
