# 10 — Адмін: налаштування системи

---

## 10.1 Загальні налаштування

### TC-AS-1.1 Редагування загальних
**Передумова:** `/admin/settings`
**Кроки:**
1. Перевірити поля: назва сайту, опис, телефон, email, адреса
2. Змінити телефон → зберегти
3. Перевірити що телефон оновився в хедері сайту
**Очікуваний результат:** Зміни відображаються на сайті
**Статус:** ⬜

---

## 10.2 Налаштування оплати

### TC-AS-2.1 Перевірка конфігурації
**Передумова:** `/admin/payment-settings`
**Кроки:**
1. Перевірити що є конфіг для: LiqPay, Monobank, WayForPay
2. Перевірити що ключі (частково замасковані) відображаються
3. Натиснути "Тест підключення" (якщо є)
**Очікуваний результат:** Статус підключення кожного провайдера
**Статус:** ⬜

---

## 10.3 Налаштування доставки

### TC-AS-3.1 Нова Пошта
**Передумова:** `/admin/delivery-settings`
**Кроки:**
1. Перевірити API ключ Нової Пошти
2. Перевірити "Тест підключення"
**Очікуваний результат:** Статус підключення OK
**Статус:** ⬜

### TC-AS-3.2 Палетна доставка
**Передумова:** `/admin/settings/pallet-delivery`
**Кроки:**
1. Перевірити налаштування: вага, тарифи, зони
2. Змінити тариф → зберегти
**Очікуваний результат:** Тарифи оновлені
**Статус:** ⬜

---

## 10.4 SMTP / Email

### TC-AS-4.1 Тест підключення SMTP
**Передумова:** `/admin/smtp-settings`
**Кроки:**
1. Перевірити: хост, порт, юзер (замаскований пароль)
2. Натиснути "Тест" → ввести email для тесту
**Очікуваний результат:** Тестовий лист надіслано
**Статус:** ⬜

---

## 10.5 Боти (Telegram / Viber)

### TC-AS-5.1 Налаштування Telegram бота
**Передумова:** `/admin/bot-settings`
**Кроки:**
1. Перевірити токен бота
2. Перевірити ID каналу / менеджера
3. Зберегти зміни
**Очікуваний результат:** Конфігурація збережена
**Статус:** ⬜

### TC-AS-5.2 Авто-відповіді бота
**Передумова:** `/admin/bot-replies`
**Кроки:**
1. Додати правило: тригер `ціна` → відповідь `Дивіться наш прайс...`
2. Зберегти
**Очікуваний результат:** Правило збережено
**Статус:** ⬜

---

## 10.6 Купони та знижки

### TC-AS-6.1 Створення купона
**Передумова:** `/admin/coupons`
**Кроки:**
1. "Додати купон"
2. Код: `TEST10`
3. Тип: процентна знижка, 10%
4. Мін. сума: 500 грн
5. Дійсний до: через місяць
6. Зберегти
**Очікуваний результат:** Купон створено
**Статус:** ⬜

### TC-AS-6.2 Застосування купона при checkout
**Передумова:** Кошик з сумою > 500 грн
**Кроки:**
1. На сторінці checkout ввести `TEST10`
2. Натиснути "Застосувати"
3. Перевірити що сума зменшилась на 10%
**Очікуваний результат:** Знижка застосована
**Статус:** ⬜

---

## 10.7 Персональні ціни

### TC-AS-7.1 Створення правила персональної ціни
**Передумова:** `/admin/personal-prices`
**Кроки:**
1. "Додати правило"
2. Користувач: обрати з пошуку
3. Тип: фіксована ціна або % знижки
4. Категорія/товар: обрати
5. Зберегти
**Очікуваний результат:** Правило збережено, юзер бачить свою ціну
**Статус:** ⬜
