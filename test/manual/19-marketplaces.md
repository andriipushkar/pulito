# 19 — Маркетплейси: OLX, Rozetka, Prom.ua, Epicentr K

> **Налаштування:** `/admin/marketplaces`
> **Передумова:** API ключі кожного маркетплейсу налаштовані

---

## 19.1 OLX

### TC-MK-1.1 Підключення OLX
**Передумова:** `/admin/marketplaces`
**Кроки:**
1. Знайти OLX у списку
2. Ввести Access Token
3. Зберегти
4. Натиснути "Тест підключення"
**Очікуваний результат:** Статус "Підключено"
**Статус:** ⬜

### TC-MK-1.2 Публікація товару на OLX
**Кроки:**
1. Обрати товар → "Опублікувати на OLX"
2. Перевірити preview: заголовок (до 70 символів), опис (до 9000), фото (до 8)
3. Підтвердити
**Очікуваний результат:** Оголошення створено, externalId збережено
**Статус:** ⬜

### TC-MK-1.3 Оновлення ціни на OLX
**Кроки:**
1. Змінити ціну товару в адмінці
2. Натиснути "Синхронізувати ціни" для OLX
**Очікуваний результат:** Ціна оновлена на OLX
**Статус:** ⬜

### TC-MK-1.4 Повідомлення OLX
**Кроки:**
1. `/admin/marketplaces` → "Повідомлення" → OLX
2. Перевірити список непрочитаних повідомлень
**Очікуваний результат:** Список тредів з ім'ям, текстом, оголошенням
**Статус:** ⬜

## 19.6 Масова синхронізація (Cron)

### TC-MK-6.1 Автоматична синхронізація цін
**Передумова:** Є опубліковані товари на маркетплейсах
**Кроки:**
1. Змінити ціну кількох товарів в адмінці
2. Дочекатися виконання cron `/api/v1/cron/sync-marketplace-prices` (або запустити вручну)
3. Перевірити що ціни оновились на маркетплейсах
**Очікуваний результат:** Ціни автоматично синхронізовано
**Статус:** ⬜

## 19.7 Instagram

### TC-MK-7.1 Публікація фото в Instagram
**Передумова:** `/admin/publications`, Instagram підключено
**Кроки:**
1. Створити публікацію з фото
2. Обрати канал "Instagram"
3. Додати текст (caption)
4. Натиснути "Опублікувати"
**Очікуваний результат:** Пост опубліковано, permalink збережено
**Статус:** ⬜

### TC-MK-7.2 Публікація каруселі Instagram
**Кроки:**
1. Створити публікацію з 2-10 фото
2. Опублікувати
**Очікуваний результат:** Карусель-пост в Instagram
**Статус:** ⬜

### TC-MK-7.3 Instagram Insights
**Передумова:** `/admin/analytics` або cron `instagram-insights`
**Кроки:**
1. Перевірити метрики: impressions, reach, profile views, followers
**Очікуваний результат:** Дані Instagram аналітики відображаються
**Статус:** ⬜
