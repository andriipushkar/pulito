## 9.1 Банери

### TC-AC-1.1 Список банерів
**Передумова:** `/admin/banners`
**Кроки:**
1. Перевірити список банерів з превью зображень
2. Перевірити порядок (drag & drop або стрілки)
**Очікуваний результат:** Банери відображаються з превью
**Статус:** ⬜

### TC-AC-1.2 Створення банера
**Кроки:**
1. Натиснути "Додати банер"
2. Завантажити зображення (JPEG, до 5MB)
3. Вказати посилання: `/catalog?category=aktsii`
4. Зберегти
5. Перевірити на головній сторінці
**Очікуваний результат:** Банер видно на головній
**Статус:** ⬜

### TC-AC-1.3 Видалення банера
**Кроки:**
1. Натиснути "Видалити" → підтвердити
**Очікуваний результат:** Банер зник з головної
**Статус:** ⬜

## 9.2 Статичні сторінки (CMS)

### TC-AC-2.1 Створення сторінки
**Передумова:** `/admin/pages`
**Кроки:**
1. "Додати сторінку"
2. Заголовок: `Доставка та оплата`
3. Slug: `dostavka-ta-oplata`
4. Контент: HTML або Markdown з описом
5. Зберегти та опублікувати
6. Перевірити на `/pages/dostavka-ta-oplata`
**Очікуваний результат:** Сторінка доступна публічно
**Статус:** ⬜

## 9.3 FAQ

### TC-AC-3.1 Додавання запитання
**Передумова:** `/admin/faq`
**Кроки:**
1. "Додати"
2. Питання: `Як відстежити замовлення?`
3. Відповідь: `Ви можете відстежити замовлення в особистому кабінеті...`
4. Категорія: `Доставка`
5. Зберегти
6. Перевірити на `/faq`
**Очікуваний результат:** Запитання з'явилось на публічній FAQ сторінці
**Статус:** ⬜

### TC-AC-3.2 Пошук у FAQ
**Передумова:** `/faq`
**Кроки:**
1. Ввести `доставка` в пошук
2. Перевірити що показуються релевантні запитання
**Очікуваний результат:** Фільтрація працює
**Статус:** ⬜

## 9.7 Канали, модерація та зворотній зв'язок

### TC-AC-7.1 Управління каналами
**Передумова:** Увійти як admin, перейти `/admin/channels`
**Кроки:**
1. Перевірити список каналів продажів
2. Натиснути "Додати канал"
3. Обрати тип каналу (веб, маркетплейс, соцмережа)
4. Налаштувати параметри каналу
5. Зберегти
**Очікуваний результат:** Канал продажів додано та налаштовано, відображається у списку
**Статус:** ⬜

### TC-AC-7.2 Правила модерації
**Передумова:** Увійти як admin, перейти `/admin/moderation`
**Кроки:**
1. Натиснути "Створити правило"
2. Обрати тип контенту (відгуки, коментарі)
3. Задати умову: містить заборонені слова / рейтинг < 2 / новий користувач
4. Задати дію: автовідхилення / на ручну модерацію / автопублікація
5. Зберегти
**Очікуваний результат:** Правило автоматичної модерації створено та працює для нового контенту
**Статус:** ⬜

### TC-AC-7.3 Логи модерації
**Передумова:** Увійти як admin, перейти на сторінку логів модерації
**Кроки:**
1. Перевірити журнал дій модерації: дата, модератор, дія, контент
2. Перевірити фільтрацію по типу дії (схвалено, відхилено, автомодерація)
3. Перевірити фільтрацію по модератору
4. Натиснути на запис — перевірити деталі
**Очікуваний результат:** Журнал дій модерації відображається з фільтрацією та деталями
**Статус:** ⬜

### TC-AC-7.4 Зворотній зв'язок
**Передумова:** Увійти як admin, перейти `/admin/feedback`
**Кроки:**
1. Перевірити список звернень клієнтів: дата, ім'я, email, тема, статус
2. Натиснути на звернення — перевірити повний текст
3. Змінити статус на "Оброблено"
4. Додати коментар/відповідь
5. Зберегти
**Очікуваний результат:** Звернення клієнтів відображаються, обробка та зміна статусу працює
**Статус:** ⬜

### TC-AC-7.5 Управління блоками головної
**Передумова:** Увійти як admin, перейти `/admin/homepage`
**Кроки:**
1. Перевірити список блоків головної сторінки (банери, товари, акції, тощо)
2. Натиснути "Додати блок" — обрати тип
3. Налаштувати контент блоку
4. Змінити порядок блоків (drag & drop або стрілки)
5. Зберегти
6. Перевірити зміни на головній сторінці `/`
**Очікуваний результат:** Блоки головної сторінки додаються, редагуються та змінюють порядок
**Статус:** ⬜
