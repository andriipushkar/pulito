# 07 — Адмін: товари, імпорт, категорії

---

## 7.1 Список товарів

### TC-AP-1.1 Перегляд списку
**Передумова:** `/admin/products`
**Кроки:**
1. Перевірити таблицю: зображення, код, назва, категорія, ціна, залишок, статус
2. Перевірити пошук за назвою/кодом
3. Перевірити фільтр за категорією
4. Перевірити сортування
**Очікуваний результат:** Повний список з фільтрацією
**Статус:** ⬜

### TC-AP-1.2 Bulk-операції
**Кроки:**
1. Обрати кілька товарів (чекбокси)
2. Зробити масову операцію: змінити статус / ціну
**Очікуваний результат:** Операція застосована до всіх обраних
**Статус:** ⬜

---

## 7.2 Редагування товару

### TC-AP-2.1 Зміна ціни
**Передумова:** `/admin/products/[id]`
**Кроки:**
1. Змінити роздрібну ціну з 100 на 120
2. Зберегти
3. Перевірити на фронті `/product/[slug]`
**Очікуваний результат:** Ціна оновлена на сайті (з затримкою ISR до 120с)
**Статус:** ⬜

### TC-AP-2.2 Завантаження зображення
**Кроки:**
1. В секції "Зображення" натиснути "Додати"
2. Обрати JPEG файл (до 5MB)
3. Зберегти
4. Перевірити що з'явились варіанти: full, medium, thumbnail, blur (WebP)
**Очікуваний результат:** Зображення оброблено через sharp, 4 варіанти
**Статус:** ⬜

### TC-AP-2.3 Видалення зображення
**Кроки:**
1. Натиснути "Видалити" на зображенні
2. Підтвердити
**Очікуваний результат:** Зображення видалено, наступне стало головним
**Статус:** ⬜

### TC-AP-2.4 SEO поля товару
**Кроки:**
1. В секції SEO заповнити: meta title, meta description
2. Зберегти
3. Перевірити через View Source на фронті
**Очікуваний результат:** Meta теги оновлені
**Статус:** ⬜

---

## 7.3 Створення товару

### TC-AP-3.1 Новий товар
**Кроки:**
1. Натиснути "Додати товар"
2. Заповнити обов'язкові поля:
   - Назва: `Тестовий порошок 1кг`
   - Код: `TEST-001`
   - Категорія: обрати з випадаючого
   - Роздрібна ціна: `95.50`
   - Оптова ціна: `75.00`
   - Залишок: `100`
3. Додати опис
4. Зберегти
**Очікуваний результат:** Товар створено, видно в каталозі
**Статус:** ⬜

---

## 7.4 Імпорт товарів

### TC-AP-4.1 Імпорт з Excel
**Передумова:** `/admin/import`
**Кроки:**
1. Завантажити шаблон Excel (якщо є кнопка)
2. Заповнити 3 рядки з тестовими товарами
3. Завантажити файл
4. Перевірити preview — які товари будуть створені/оновлені
5. Підтвердити імпорт
**Очікуваний результат:** Товари імпортовано, показано результат (створено/оновлено/помилки)
**Статус:** ⬜

### TC-AP-4.2 Імпорт з помилками
**Кроки:**
1. Завантажити файл з невалідними даними (пуста ціна, дубль коду)
2. Перевірити preview з помилками
**Очікуваний результат:** Помилки показані порядково, імпорт не відбувся
**Статус:** ⬜

### TC-AP-4.3 Логи імпорту
**Кроки:**
1. Перейти на `/admin/import` → вкладка "Логи"
2. Перевірити записи попередніх імпортів
**Очікуваний результат:** Історія імпортів з деталями
**Статус:** ⬜

## 7.5 Категорії

### TC-AP-5.1 Створення категорії
**Передумова:** `/admin/categories`
**Кроки:**
1. Натиснути "Додати"
2. Назва: `Тестова категорія`
3. Slug: автогенерація
4. Зберегти
**Очікуваний результат:** Категорія створена, видна в фільтрах каталогу
**Статус:** ⬜

### TC-AP-5.2 Об'єднання категорій
**Кроки:**
1. Обрати дві категорії
2. Натиснути "Об'єднати"
3. Обрати цільову категорію
**Очікуваний результат:** Товари перенесено, стара категорія видалена
**Статус:** ⬜
