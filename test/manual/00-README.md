1. Відкрити файл відповідного блоку
2. Виконувати тести послідовно (вони залежать один від одного)
3. Ставити статус ✅ або ❌
4. При ❌ — описати що пішло не так
5. Скріншоти складати в `/test/screenshots/` з назвою тест-кейсу
