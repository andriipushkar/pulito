# Налаштування Telegram-бота

Telegram-бот використовується для:
- Сповіщень про нові замовлення
- Сповіщень про зміну статусів
- Зв'язку з клієнтами (авто-відповіді)
- Відправки повідомлень в канал магазину

## Крок 1 — Створити бота через @BotFather

1. Відкрийте Telegram і знайдіть **@BotFather**
2. Надішліть команду `/newbot`
3. Введіть назву бота, наприклад: `Clean Shop Бот`
4. Введіть юзернейм бота, наприклад: `clean_shop_bot` (має закінчуватись на `bot`)
5. BotFather надішле вам **токен** вигляду: `7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

> **Збережіть токен** — він потрібен для .env.

## Крок 2 — Додаткові налаштування бота

Надішліть @BotFather наступні команди:

```
/setdescription — опис бота (видно при першому відкритті)
/setabouttext — короткий опис (видно в профілі)
/setuserpic — завантажте аватарку магазину
/setcommands — встановіть команди:
```

Для `/setcommands` надішліть:

```
start - Почати роботу з ботом
help - Допомога
status - Статус мого замовлення
```

## Крок 3 — Знайти Chat ID менеджера

Це ID чату, куди бот буде надсилати сповіщення про замовлення.

**Спосіб 1 — через @userinfobot:**

1. Знайдіть в Telegram бота **@userinfobot**
2. Надішліть йому будь-яке повідомлення
3. Він покаже ваш `Id` — це ваш Chat ID

**Спосіб 2 — через getUpdates API:**

1. Надішліть будь-яке повідомлення вашому боту
2. Відкрийте в браузері:

```
https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates
```

3. Знайдіть у відповіді `"chat":{"id": XXXXXXX}` — це Chat ID

## Крок 4 — Знайти Channel ID (для каналу сповіщень)

Якщо хочете надсилати сповіщення в канал:

1. Створіть канал у Telegram (або використайте існуючий)
2. Додайте бота як **адміністратора** каналу (з правом відправки повідомлень)
3. Надішліть будь-яке повідомлення в канал
4. Відкрийте:

```
https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates
```

5. Знайдіть `"chat":{"id": -100XXXXXXXXXX}` — це Channel ID (починається з `-100`)

> Якщо `getUpdates` повертає порожній масив — надішліть повідомлення заново і перевірте ще раз.

## Крок 5 — Встановити Webhook

Webhook потрібен, щоб бот отримував повідомлення від користувачів.

```bash
curl -X POST "https://api.telegram.org/bot<YOUR_BOT_TOKEN>/setWebhook" \
  -H "Content-Type: application/json" \
  -d '{"url": "https://yourdomain.com/api/webhooks/telegram"}'
```

Очікувана відповідь:

```json
{"ok": true, "result": true, "description": "Webhook was set"}
```

Перевірити поточний webhook:

```bash
curl "https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getWebhookInfo"
```

## Крок 6 — Додати змінні в .env

```env
TELEGRAM_BOT_TOKEN=7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
TELEGRAM_CHANNEL_ID=-100XXXXXXXXXX
TELEGRAM_MANAGER_CHAT_ID=XXXXXXXXX
```

Перезапустіть додаток:

```bash
# Локально
npm run dev

# Production
pm2 reload clean-shop
```

## Крок 7 — Перевірити роботу

1. Відкрийте чат з вашим ботом у Telegram
2. Надішліть `/start`
3. Бот має відповісти вітальним повідомленням

Перевірте сповіщення:
1. Зробіть тестове замовлення на сайті
2. Менеджер має отримати повідомлення в особистий чат
3. Канал має отримати повідомлення (якщо налаштовано)

## Крок 8 — Налаштування в адмін-панелі

Після успішного підключення:

1. Зайдіть в адмін-панель → **Налаштування** → **Telegram**
2. Налаштуйте **авто-відповіді** — шаблони повідомлень для клієнтів
3. Налаштуйте **розклад сповіщень** — години роботи менеджера
4. Налаштуйте **типи сповіщень** — які події надсилати в канал/менеджеру

## Усунення проблем

### Бот не відповідає

```bash
# Перевірте, що webhook встановлений
curl "https://api.telegram.org/bot<TOKEN>/getWebhookInfo"

# Перевірте, що APP_URL доступний ззовні
curl https://yourdomain.com/api/webhooks/telegram
```

### Помилка `401 Unauthorized`

Перевірте, що `TELEGRAM_BOT_TOKEN` в .env вірний і без зайвих пробілів.

### Бот не надсилає в канал

- Переконайтеся, що бот є адміністратором каналу
- Перевірте `TELEGRAM_CHANNEL_ID` (має починатися з `-100`)
