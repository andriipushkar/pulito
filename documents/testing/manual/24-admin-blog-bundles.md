# 24 — Адмін: блог, категорії блогу, бандли

---

## 24.1 Список блог-постів

### TC-BB-1.1 Таблиця блог-постів
**Передумова:** `/admin/blog`
**Кроки:**
1. Перевірити таблицю: заголовок, статус (чернетка/опубліковано), дата, автор
2. Перевірити пагінацію
**Очікуваний результат:** Таблиця з усіма постами, пагінація працює
**Статус:** ⬜

### TC-BB-1.2 Пошук блог-постів
**Кроки:**
1. Ввести ключове слово в поле пошуку
2. Перевірити результати
**Очікуваний результат:** Відображаються тільки пости що містять ключове слово
**Статус:** ⬜

### TC-BB-1.3 Фільтрація за статусом
**Кроки:**
1. Обрати фільтр "Чернетки"
2. Перевірити що відображаються тільки чернетки
3. Обрати "Опубліковані"
4. Перевірити що відображаються тільки опубліковані
**Очікуваний результат:** Фільтрація працює коректно для кожного статусу
**Статус:** ⬜

---

## 24.2 Створення блог-посту

### TC-BB-2.1 Створення посту — основні поля
**Передумова:** `/admin/blog/new`
**Кроки:**
1. Ввести заголовок
2. Перевірити автогенерацію slug з заголовка
3. Заповнити контент у rich text редакторі
4. Завантажити обкладинку
5. Обрати категорію
**Очікуваний результат:** Всі поля заповнюються коректно, slug генерується автоматично
**Статус:** ⬜

### TC-BB-2.2 SEO поля блог-посту
**Кроки:**
1. Заповнити SEO title та meta description
2. Перевірити обмеження довжини (title ≤ 60, description ≤ 160)
3. Перевірити превʼю у стилі Google
**Очікуваний результат:** SEO поля валідуються, превʼю відображається
**Статус:** ⬜

### TC-BB-2.3 Збереження як чернетка
**Кроки:**
1. Заповнити основні поля
2. Натиснути "Зберегти як чернетку"
**Очікуваний результат:** Пост збережено зі статусом "Чернетка", не видимий на сайті
**Статус:** ⬜

### TC-BB-2.4 Публікація посту
**Кроки:**
1. Заповнити всі обовʼязкові поля
2. Натиснути "Опублікувати"
**Очікуваний результат:** Пост опубліковано, видимий на сайті за URL /blog/{slug}
**Статус:** ⬜

## 24.3 Редагування та видалення блог-посту

### TC-BB-3.1 Редагування опублікованого посту
**Передумова:** Існує опублікований пост
**Кроки:**
1. Відкрити пост на редагування
2. Змінити заголовок та контент
3. Натиснути "Оновити"
**Очікуваний результат:** Зміни збережено, оновлений контент відображається на сайті
**Статус:** ⬜

### TC-BB-3.2 Превʼю посту перед публікацією
**Кроки:**
1. Внести зміни в пост
2. Натиснути "Превʼю"
**Очікуваний результат:** Відкривається превʼю сторінка з актуальним контентом
**Статус:** ⬜

### TC-BB-3.3 Видалення блог-посту
**Кроки:**
1. Натиснути "Видалити" біля посту
2. Підтвердити в діалозі
**Очікуваний результат:** Пост видалено, сторінка /blog/{slug} повертає 404
**Статус:** ⬜

## 24.4 Категорії блогу

### TC-BB-4.1 Список категорій
**Передумова:** `/admin/blog/categories`
**Кроки:**
1. Перевірити таблицю: назва, slug, опис, кількість постів
**Очікуваний результат:** Всі категорії відображаються з кількістю привʼязаних постів
**Статус:** ⬜

### TC-BB-4.2 Створення категорії
**Кроки:**
1. Натиснути "Додати категорію"
2. Заповнити: назва, slug (автогенерація), опис
3. Зберегти
**Очікуваний результат:** Категорія створена, доступна для вибору при створенні постів
**Статус:** ⬜

### TC-BB-4.3 Редагування категорії
**Кроки:**
1. Натиснути "Редагувати" біля категорії
2. Змінити назву та опис
3. Зберегти
**Очікуваний результат:** Зміни збережено, оновлена назва відображається в постах
**Статус:** ⬜

### TC-BB-4.4 Видалення категорії
**Кроки:**
1. Натиснути "Видалити" біля категорії
2. Перевірити попередження якщо є привʼязані пости
3. Підтвердити
**Очікуваний результат:** Категорія видалена, пости відвʼязані від категорії
**Статус:** ⬜

## 24.5 Бандли (набори товарів)

### TC-BB-5.1 Список бандлів
**Передумова:** `/admin/bundles`
**Кроки:**
1. Перевірити таблицю: назва бандла, кількість товарів, ціна, статус (активний/неактивний)
2. Перевірити пагінацію
**Очікуваний результат:** Таблиця з усіма бандлами, коректна пагінація
**Статус:** ⬜

### TC-BB-5.2 Створення бандла
**Кроки:**
1. Натиснути "Створити бандл"
2. Ввести назву та опис
3. Додати товари через пошук та вибір
4. Встановити ціну бандла (порівняння з сумою окремих товарів)
5. Встановити % знижки
6. Завантажити зображення
7. Зберегти
**Очікуваний результат:** Бандл створено, ціна розрахована з урахуванням знижки
**Статус:** ⬜

### TC-BB-5.3 Редагування бандла
**Передумова:** Існуючий бандл
**Кроки:**
1. Відкрити бандл на редагування
2. Додати новий товар, видалити існуючий
3. Змінити ціну та активувати/деактивувати
4. Зберегти
**Очікуваний результат:** Зміни збережено, склад бандла оновлено
**Статус:** ⬜

### TC-BB-5.4 Видалення бандла
**Кроки:**
1. Натиснути "Видалити" біля бандла
2. Перевірити попередження якщо бандл є в замовленнях
3. Підтвердити
**Очікуваний результат:** Бандл видалено, існуючі замовлення не порушені
**Статус:** ⬜
