## 1. НАВІГАЦІЯ ТА ХЕДЕР

### TC-M-1.1 Логотип та перехід на головну

**Передумова:** Відкрити будь-яку сторінку, наприклад `/catalog`
**Кроки:**

1. Подивитися на логотип у верхньому лівому куті
2. Бейдж "П" — gradient background, rounded-xl, shadow-brand
3. "Поро" + "шок" — градієнтний текст (від primary до primary-light)
4. Натиснути на логотип
   **Очікуваний результат:** Перехід на головну `/`
   **Статус:** ⬜

### TC-M-1.2 Top Bar — посилання та соцмережі

**Передумова:** Головна сторінка
**Кроки:**

1. У верхній частині є рядок з "Зв'язок" та "Питання"
2. Іконки соцмереж: Telegram, Viber, Instagram, Facebook, TikTok (розмір 14px)
3. Натиснути "Зв'язок" → `/contacts`
4. Натиснути "Питання" → `/faq`
5. Проскролити вниз (>80px) → TopBar зникає
   **Очікуваний результат:** TopBar функціональний, ховається при скролі
   **Статус:** ⬜

### TC-M-1.3 Сповіщення в хедері

**Передумова:** Авторизований з непрочитаними сповіщеннями
**Кроки:**

1. У хедері — між логотипом та кнопкою "Зателефонувати"
2. Іконка дзвіночка (Bell) з червоним бейджем кількості
3. Кількість оновлюється кожні 60 секунд
4. Натиснути → `/account/notifications`
   **Очікуваний результат:** Перехід на сторінку сповіщень
   **Статус:** ⬜

### TC-M-1.4 Кнопка "Зателефонувати"

**Передумова:** Будь-яка сторінка
**Кроки:**

1. У хедері — кнопка "Зателефонувати" з іконкою телефону (синій фон primary/10)
2. Натиснути → відкрився номеронабирач з номером магазину (+38 (000) 123-45-67)
   **Очікуваний результат:** tel: посилання працює
   **Статус:** ⬜

### TC-M-1.5 Пошук — поле під хедером

**Передумова:** Будь-яка сторінка
**Кроки:**

1. Під хедером — поле пошуку (placeholder "Пошук товарів...")
2. Натиснути на поле → фулскрін режим пошуку (< 640px)
3. При фокусі з'являється клавіатура
4. Ввести "порошок" (мін. 2 символи) → через 300мс з'являються результати:
   - Категорії + товари (мініатюра, назва, код, ціна)
5. Enter → `/catalog?search=порошок`
   **Очікуваний результат:** Пошук з автопідказками у фулскрін-режимі
   **Статус:** ⬜

### TC-M-1.6 Пошук за кодом

**Передумова:** Поле пошуку
**Кроки:**

1. Ввести артикул товару → Enter
   **Очікуваний результат:** Знайдено товар за кодом
   **Статус:** ⬜

### TC-M-1.7 Порожній пошук

**Передумова:** Будь-яка сторінка, Chrome DevTools, ширина 390px
**Кроки:**

1. Ввести "фівапролджфівапролдж" → Enter
   **Очікуваний результат:** Каталог з іконкою пошуку + "Товарів не знайдено" + посилання скидання
   **Статус:** ⬜

### TC-M-1.8 Таб-бар — структура

**Передумова:** Будь-яка сторінка
**Кроки:**

1. Внизу — фіксований таб-бар з 5 іконками:
   - Головна

### TC-M-1.10 Таб-бар — каталог (bottom sheet)

### TC-M-1.11 Таб-бар — кошик з бейджем

3. Натиснути → `/cart`
   **Очікуваний результат:** Перехід у кошик
   **Статус:** ⬜

### TC-M-1.12 Таб-бар — обране з бейджем

**Передумова:** Авторизований, є товари в обраному
**Кроки:**

1. "Обране" — синій бейдж із кількістю
2. Натиснути → `/account/wishlist`
   **Очікуваний результат:** Перехід до обраного
   **Статус:** ⬜

### TC-M-1.13 Таб-бар — профіль зі сповіщеннями

**Передумова:** Авторизований з непрочитаними сповіщеннями
**Кроки:**

1. "Профіль" — червоний бейдж із кількістю сповіщень
2. Натиснути → `/account`
3. Неавторизований → `/auth/login`
   **Очікуваний результат:** Правильний маршрут залежно від стану

### TC-M-1.14 Таб-бар — приховування при скролі

**Передумова:** Головна або каталог
**Кроки:**

1. Швидко скролити вниз
2. Скролити вгору
   **Очікуваний результат:** Таб-бар плавно з'являється
   **Статус:** ⬜

### TC-M-1.15 Bottom sheet — закриття

**Передумова:** Відкритий bottom sheet каталогу
**Кроки:**

1. Натиснути X (правий верхній кут) → закрився
2. Відкрити знову → натиснути на затемнену область → закрився
3. Відкрити знову → Escape → закрився
4. Body scroll заблокований поки sheet відкритий
5. Focus trap працює (Tab не виходить за межі sheet)
   **Очікуваний результат:** Всі методи закриття працюють, доступність забезпечена
   **Статус:** ⬜

### TC-M-1.16 Таб-бар на 320px (iPhone SE)

**Передумова:** Chrome DevTools → 320px
**Кроки:**

1. Всі 5 підписів видимі, не обрізаються
2. Іконки не накладаються одна на одну
3. Кнопка кошика (центральна) не виходить за межі
   **Очікуваний результат:** Таб-бар коректний на вузькому екрані
   **Статус:** ⬜

## 2. ГОЛОВНА СТОРІНКА

### TC-M-2.1 Банер — відображення

**Передумова:** Головна сторінка
**Кроки:**

1. Банер має пропорції 5:2 (aspect-[5/2])
2. Зображення заповнює весь контейнер (object-cover)
3. rounded-3xl та shadow
4. Текстовий блок: glass-dark стиль, заголовок + CTA-кнопка
5. Підзаголовок прихований на мобільній (hidden, block на sm+)
6. Прогрес-індикатори внизу — тонкі лінії (НЕ крапки)
   **Очікуваний результат:** Банер відображається коректно
   **Статус:** ⬜

### TC-M-2.2 Банер — свайп та автопрокрутка

**Передумова:** Головна, банер видимий
**Кроки:**

1. Зачекати 5с → банер автоматично переключився, прогрес-бар анімувався
2. Свайпнути банер вліво → наступний слайд
3. Свайпнути вправо → попередній слайд
   **Очікуваний результат:** Touch-свайп працює плавно
   **Статус:** ⬜

### TC-M-2.3 Банер — клік

**Передумова:** Головна
**Кроки:**

1. Натиснути на банер (на CTA-кнопку або зображення)
   **Очікуваний результат:** Перехід за посиланням (buttonLink)
   **Статус:** ⬜

### TC-M-2.4 Категорії — сітка (тільки мобільна/планшет)

**Передумова:** Головна, блок категорій (hidden на lg+)
**Кроки:**

1. **Мобільна:** Горизонтальний скрол (stories-стиль, 72px карточки)
2. **Планшет (sm+):** Горизонтальний скрол, більші карточки (88-96px, 120-130px wide)
3. Кожна карточка: іконка/зображення + назва (2-line clamp) + кількість товарів
4. Натиснути → `/catalog?category=slug`
   **Очікуваний результат:** Категорії відображаються, посилання працюють
   **Статус:** ⬜

### TC-M-2.5 Каруселі товарів

**Передумова:** Головна, є акційні/нові/популярні товари
**Кроки:**

1. Блоки: "Акційні товари", "Новинки", "Хіти продажів" (якщо є дані)
2. Заголовок + "Дивитися все" (посилання)
3. Свайп каруселі вліво/вправо
4. Натиснути на картку → `/product/slug`
5. Натиснути "В кошик" на картці → toast, бейдж кошика у таб-барі +1
   **Очікуваний результат:** Каруселі працюють з touch
   **Статус:** ⬜

### TC-M-2.6 Блок брендів — автоскрол

**Передумова:** Головна, "Наші бренди"
**Кроки:**

1. Заголовок із золотим підкресленням
2. Бренди прокручуються горизонтально автоматично (30s, infinite)
3. Натиснути на бренд → `/catalog?brand=slug`
   **Очікуваний результат:** Автоскрол та посилання працюють
   **Статус:** ⬜

### TC-M-2.7 SEO-текст

**Передумова:** Головна, проскролити до кінця
**Кроки:**

1. Блок "Інтернет-магазин побутової хімії Порошок"
2. Gradient фон (primary-50/60 → white) + тонка рамка
3. Текст max-w-3xl, читабельний
   **Очікуваний результат:** SEO-блок оформлений та не розтягнутий
   **Статус:** ⬜

### TC-M-2.8 Нещодавно переглянуті

**Передумова:** Переглянути 2-3 товари, повернутися на головну
**Кроки:**

1. Блок "Нещодавно переглянуті" з каруселлю
2. Якщо нічого не переглядали — блок не показується (self-hiding)
   **Очікуваний результат:** Відображає реальну історію перегляду
   **Статус:** ⬜

### TC-M-2.9 Порожні блоки не відображаються

**Передумова:** Головна
**Кроки:**

1. Блоки керуються з БД (enabled/disabled)
2. Якщо немає акційних товарів — блок не рендериться
3. Між видимими блоками немає порожніх просторів
   **Очікуваний результат:** Тільки блоки з контентом, без "дірок"
   **Статус:** ⬜
