## Full Description

### What is this product?

Clean Shop is a **complete, production-ready e-commerce platform** purpose-built for the FMCG and household chemicals market. It's not a WordPress template or a Shopify clone — it's a **175,000-line TypeScript codebase** with 92 business logic services, 307 REST API endpoints, and 20+ integrations covering the entire Ukrainian e-commerce ecosystem.

The platform works as both a **standalone B2B/B2C online store** and a **white-label SaaS** — with built-in multi-tenancy, billing (plans/invoices), custom domains, and feature flags.

### What makes it special?

**B2B Pricing Engine** — 3 wholesale tiers, per-client personal pricing, volume discounts, bulk ordering (CSV/Excel import), commercial proposal PDF generation. This is not a basic WooCommerce shop — it's built for distributors handling hundreds of SKUs and dozens of wholesale clients.

**Telegram & Viber Bots** — Not plugins. Native bots with full catalog browsing, product search, account linking, order status notifications, and FAQ. Your customers order through the bot without ever opening a browser.

**4 Ukrainian Marketplaces** — Prom.ua, Rozetka, OLX, and Epicentr K synchronization. Prices, stock, orders, and returns sync automatically via cron jobs every 6 hours.

**18-Tab Analytics Dashboard** — ABC analysis, RFM segmentation, LTV, cohorts, sales funnel, churn prediction, geography, channel analytics, stock analysis, price tracking. 13 types of PDF/Excel reports. Configurable alerts via Telegram/email.

**97% Test Coverage** — 661 unit test files, 71 E2E specs (Playwright), 110 Storybook stories, 1,217 manual test cases with step-by-step instructions. Every API endpoint, every service, every component has a test. This is enterprise-level quality.

### Tech Stack

- **Frontend:** Next.js 16 (App Router, RSC), React 19, TypeScript 5.9, Tailwind CSS 4
- **Backend:** PostgreSQL 16, Prisma 7.4 (94 models), Redis 7, BullMQ, PgBouncer
- **Search:** Typesense 27 (fulltext, typo tolerance, instant search)
- **Infrastructure:** Docker (multi-stage, ~60MB), GitHub Actions CI/CD, PM2, Nginx
- **Monitoring:** Sentry, Axiom, UptimeRobot
- **37 dependencies, 32 dev dependencies** — lean, no bloat
